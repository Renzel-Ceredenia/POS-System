FAQ

Frequently Asked Questions

What database does this use? ▾

Firebase Realtime Database hosted at pos-electives-default-rtdb.firebaseio.com. All reads and writes go directly to Firebase — no local storage, no mock data.

How does authentication work? ▾

Firebase Email/Password Authentication. The admin account is created automatically on first run. Passwords are handled entirely by Firebase — never stored in plain text.

Can multiple users log in simultaneously? ▾

Yes. Firebase Realtime Database supports concurrent connections. All changes sync instantly across all sessions.

Does this system calculate tax? ▾

No. By design, this POS system does not calculate tax. The formula is: Total = (Price × Quantity) − Discount.

How do I create new user accounts? ▾

Log in as Admin → Settings → User Management. You can create, edit, disable, and reset passwords for any account from there.

Is the system mobile-friendly? ▾

Yes. The entire interface is fully responsive. The sidebar collapses on mobile and all layouts stack gracefully for smaller screens.
